Message d’erreur de connexion

Quand vous n’avez pas spécifié d’autorité parente, les utilisateurs ne peuvent se connecter qu’à travers le système SSO via l’application distante et ne peuvent pas se connecter directement depuis le formulaire d’accès de Mahara.

Entrez un message qui s’affichera lorsqu’un utilisateur tentera de se connecter directement depuis le formulaire d’accès de Mahara. Ce message doit contenir les informations nécessaires pour renvoyer l’utilisateur à l’application distante afin d’accéder à Mahara par SSO.
