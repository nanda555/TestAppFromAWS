Simple PHP SAML Camino de configuracion

La ruta de acceso completa al directorio de configuracion que se utilizara para este proveedor de servicios (SP) Simple PHP SAML. por ejemplo: si la ruta de acceso completa para el archivo config.php es / some_path_to/simplesamlphp_1_3/config/config.php a continuacion, este valor debe ser / some_path_to/simplesamlphp_1_3/config /
