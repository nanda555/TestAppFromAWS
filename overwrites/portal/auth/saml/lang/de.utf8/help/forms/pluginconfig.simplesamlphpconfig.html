Einfacher SAML PHP Konfiguration Pfad

Der vollständige Pfad zum Konfigurationsverzeichnis, das für diesen Service Provider (SP) Simple SAML PHP benutzt werden soll. z.B.: Wenn der vollständige Pfad zur config.php is /some_path_to/simplesamlphp_1_3/config/config.php ist dann sollte in diesem Feld /some_path_to/simplesamlphp_1_3/config/ eingetragen werden
