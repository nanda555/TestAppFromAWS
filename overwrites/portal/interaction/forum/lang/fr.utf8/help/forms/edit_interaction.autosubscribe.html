Abonner automatiquement les utilisateurs

Si vous désirez que vos utilisateurs soient automatiquement abonnés aux contributions de ce forum quand ils le rejoignent pour la première fois, activez cette option.

Une fois abonnés, les utilisateurs recevront un message à chaque fois qu'une nouvelle contribution sera créée dans le forum.

Les utilisateurs ont toujours la possibilité d'arrêter leur abonnement et de stopper l'envoi automatiquement des nouvelles contributions du forum.
