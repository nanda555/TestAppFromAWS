Benachrichtigungen

Sie können wählen, wie Sie Benachrichtigungen erhalten für jede Aktivität aus.

Benachrichtigung Typen

Aktivitätsprotokoll: Ihre Benachrichtigungen Bereich wird eine Liste der Aktivität eingegangenen Meldungen.

Email: Ihre primäre E-Mail-Adresse wird eine E-Mail jedes Mal, wenn die Aktivität auftritt.

Email verdauen: Ihre primäre E-Mail-Adresse wird eine E-Mail jeden Tag mit einer Liste der letzten 24 Stunden Aktivitäten.

Note: Wenn Sie eine E-Mail-Option alle Aktivitäten werden auch in Ihren Mitteilungen aufgezeichnet werden log, wie sie geschehen. Sie werden alle markierten, wie bereits zu lesen. Diese Meldungen automatisch verfallen und werden von Ihrem log nach 60 Tagen entfernt.

Leistungsarten

Systemmeldungen: vom System automatisch generiert oder an Sie gesendet werden von einem der Website-Administratoren.

Nachrichten von anderen Nutzern: an Sie direkt von anderen Benutzern der Website.

Rückkopplung: Mitteilung über jedes Feedback oder Kommentare auf Seiten, Artefakte oder Blogs in Ihrem ePortfolio erhielt.

Watchlist: Mitteilung der Aktivität auf jedem Artefakt, Seite, Gruppe oder Blog Sie überwachen.

Die Seite wurde neu Zugang: Benachrichtigung, die Sie oder einer Ihrer Gruppen haben Zugang zu einer neuen oder bestehenden nicht-öffentlichen Seite.
