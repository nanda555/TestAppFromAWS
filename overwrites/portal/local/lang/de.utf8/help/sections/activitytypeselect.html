Leistungsarten

Wählen Sie aus diesem Dropdown-Menü die Meldungsart, die Sie anzeigen möchten.

Neues Forum Beitrag: Mitteilung über einen neuen Beitrag zu einem Thema im Forum die Sie abonniert haben

Rückkopplung: Mitteilung über jedes Feedback oder Kommentare auf Seiten, Artefakte oder Blogs in Ihrem ePortfolio erhielt.

Sammelmeldung: Mitteilung an Gruppe Besitzer, wenn neue Seiten zu einer Gruppe eingereicht werden.

Systemmeldung: vom System automatisch generiert oder an Sie gesendet werden von einem der Website-Administratoren.

Nachricht von anderen Nutzern: an Sie direkt von anderen Benutzern der Website.

Die Seite wurde neu Zugang: Benachrichtigung, die Sie oder einer Ihrer Gruppen haben Zugang zu einer neuen oder bestehenden nicht-öffentlichen Seite.

Watchlist: Mitteilung der Aktivität auf jedem Artefakt, Seite, Gruppe oder Blog Sie überwachen.
