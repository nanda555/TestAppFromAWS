Datei hochladen

Sie können Ihrem Blog Posting beliebige Dateien hinzufügen. Die einem Blog Posting hinzugefügten Dateien werden automatisch innerhalb Ihres Dateiablagesystems in ein Verzeichnis ‘Blogfiles’ abgespeichert und Ihrem Kontingent angerechnet.

Wenn Sie eine Datei hochladen,das Posting aber verwerfen oder den Blogbereich ohne Speicherung verlassen, so werden diese Dateien auch aus Ihrem Dateiablagesystem gelöscht.

Wenn Sie eine Datei hochgeladen haben, so können Sie den Bild einfügen/bearbeiten Button im HTML-Editor benutzen, um das Bild in Ihr Posting einzubetten. Sie können die Größe, die Anordnung, die  Einfassung und den Platz um das Bild mit den bereitgestellten Feldern anpassen.

Sie können ebenso ein Bild als Link von einer anderen Website einbetten. Dies geschieht entweder durch die Angabe der  URL des Bildes oder durch  ‘drag and drop’ von der Website in den HTML Editor. Stellen Sie bitte sicher, dass Sie die entsprechende Copyright Genehmigung für die Benutzung des Bildes einholen.

Zur Erzielung bester Ergebnisse sollte das Bild nicht größ;er als  400x400 Pixels sein und weniger als 5 Pixels Rand an jeder Seite aufweisen.
