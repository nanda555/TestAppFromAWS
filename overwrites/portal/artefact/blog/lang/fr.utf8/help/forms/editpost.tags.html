Tags

Vous pouvez ajouter des Tags aux productions ou aux pages que vous créez dans Mahara. Les Tags vous permettent d’ajouter des mots-clés, des étiquettes qui décrivent vos productions et vos pages. Les Tags permettent aussi la création d’un index afin de retrouver et classer vos informations.

Pour associer plusieurs Tags à une production ou à une page, séparez-les à l’aide d’une virgule. Par exemple — « Validation, Semestre 2, Biologie ».

Cliquer sur Afficher mes tags génère une liste de Tags que vous avez déjà créés ainsi que le nombre de fois que vous les avez utilisés. Cliquez sur un Tag pour l’ajouter à la liste des Tags pour la production ou la page courante.

Utiliser Chercher dans mon portfolio pour trouver toutes les productions et les pages qui sont associées à un Tag particulier. Vous pouvez de ce fait facilement créer un système de filtre ou un index pour votre portfolio.
