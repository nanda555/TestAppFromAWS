Externe Gallerie URL

Wenn Sie eine externe Gallerie einbetten wollen, müssen Sie den Link der externen Gallerie kopieren und in den Textbereich eintragen.

Die folgenden externen Gallerien können eingebettet werden:

  • Flickr, z.B. http://www.flickr.com/photos/12345678@N01/sets/98765432123456789/
  • Picasa, z.B. http://picasaweb.google.com/user.name/AlbumName#
  • Panoramio, z.B. http://www.panoramio.com/user/123456
  • Photobucket, z.B. http://s1234.photobucket.com/albums/ee123/username/ or http://s1234.photobucket.com/albums/ee123/username/AlbumName/
  • Windows Live Foto Gallerie, z.B. http://cid-123456ab7890cd12.photos.live.com/self.aspx/AlbumName/photo.jpg

Achtung:

  • Damit Sie Flickr einbetten können, brauchen Sie von den Administrator/innen einen gültigen API Schlüssel.
  • Damit Sie Photobucket einbetten können, brauchen Sie von den Administrator/innen einen API Schlüssel und einen geheimen API Schlüssel.
  • Damit die Windows Live Foto-Gallerie einwandfrei funktioniert, muss der Link eines Fotos im Album kopiert und in das Textfeld eingetragen werden. Dann wird das ganze Album angezeigt.
