URL de la galerie externe

Pour inclure une galerie d’images externe, vous devez copier le lien vers cette galerie et le coller dans la zone de texte.

Il est possible d’inclure les galeries externes suivantes :

  • Flickr, ex : http://www.flickr.com/photos/12345678@N01/sets/98765432123456789/
  • Picasa, ex : http://picasaweb.google.com/nom.utilisateur/NomAlbum#
  • Panoramio, ex : http://www.panoramio.com/utilisateur/123456
  • Photobucket, ex : http://s1234.photobucket.com/albums/ee123/nomutilisateur/ or http://s1234.photobucket.com/albums/ee123/nomutilisateur/NomAlbum/
  • Galerie de photos de Windows Live, ex : http://cid-123456ab7890cd12.photos.live.com/self.aspx/NomAlbum/photo.jpg

Note:

  • Pour utiliser le service Flickr, l’administrateur du site doit vous fournir une clé API valide.
  • Pour utiliser le service Photobucket, l’administrateur du site doit vous fournir une clé API et une API secrète valides.
  • Pour que la galerie de photo de Windows Live fonctionne correctement, le lien vers l’une des photos de l’album doit être copié et collé dans la zone de texte afin d’accéder à la totalité de l’album.
