Tags

Vous pouvez ajouter des tags ou mots-clés, à chaque production et pages que vous créez dans Mahara. Les tags vous permettent d'ajouter des mots ou des ensembles de mots qui définissent une production ou une page, puis de créer un index de tous les tags utilisés.

Pour associer plusieurs tags à une même production ou une même page, séparez chaque tag d'une virgule. Par exemple : « Evaluation, Semestre 2, Gestion ».

En cliquant sur « Afficher mes tags » vous générez une liste de tous les tags que vous avez déjà utilisés suivis du nombre de fois que vous utilisez chacun. Vous pouvez alors cliquer sur un ou plusieurs tags de la liste pour l'utiliser dans votre production ou page courante.

Quand vous utilisez l'outil « Chercher dans mon portfolio » sur la page « Mon portfolio », vous pouvez retrouver des pages ou des productions en utilisant les tags comme mots-clés. Vous créez ainsi une liste indexée pour votre portfolio.
