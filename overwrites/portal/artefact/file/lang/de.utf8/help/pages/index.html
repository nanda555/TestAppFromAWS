Dateien

Der Arbeitsbereich Dateien ist ein Ablagesystem für Ihre Verzeichnisse und Dateien, die Sie innerhalb Ihres Portfolio benutzen können.

Die Anzahl der Dateien, die Sie in das Ablagesystem hochladen können, wird durch Ihr Speicherplatzlimit bestimmt. Ihr zur Verfügung stehendes Kontingent wird in der rechten Bildschirmhälfte angezeigt.

Aus Sicherheitsgründen kann es sein, dass die Website Administrator/innen das Hochladen einiger Dateitypen deaktiviert haben.

Sie können mit der Drag&Drop-Funktion Ihre Dateien und Verzeichnisse verwalten.
