formato

Il formato ovviamente determina il layout della pagina del corso.

  • Formato settimanale - La pagina del corso è organizzato in sezioni settimanali, con la prima settimana a partire dalla data di inizio del corso.
  • argomenti formato - La pagina del corso è organizzato in sezioni.
  • formato sociali - Un forum è visualizzato nella pagina del corso.
  • SCORM formato - Per la visualizzazione di un pacchetto SCORM nella prima sezione della pagina del corso (in alternativa all'uso del SCORM / AICC modulo)
