Courriel

Bien que l’adresse de courriel que vous avez donnée lors de votre enregistrement est désormais votre adresse principale, vous pouvez avoir plusieurs autres adresses enregistrées sur ce site. Utilisez l’adresse de courriel que vous souhaitez dans votre profil.

Pour ajouter une nouvelle adresse de courriel, cliquez sur Ajouter une adresse de courriel. Quand vous « Enregistrez le profil », un message de confirmation vous est envoyé à cette nouvelle adresse pour validation..

Note : Le lien que vous recevez n’est valide que 24 heures. Si vous laissez ce délai s’écouler, vous devez répéter la procédure d’enregistrement.
