Mitgliedschaften

Mit dem  Hinzufügen Button können Sie einen Eintrag in Ihren Mitgliedschaften Bereich einfügen. Ihre Mitgliedschaften Einträge werden in umgekehrter zeitlicher Reihenfolge angezeigt.

Sichern
Stellen Sie sicher, dass Ihr Eintrag gesichert wird, bevor Sie den Mitgliedschaften Bereich verlassen.

Abbruch
Wenn Sie Ihren neuen Eintrag verwerfen wollen, können Sie mit  Abbruch das Mitgliedschaften Formular verlassen. Beachten Sie bitte, dass Ihr Eintrag dann nicht gespeichert wird.

Nach dem Speichern wird Ihr Mitgliedschaften Eintrag in tabellarischer Form angezeigt. Mit Bearbeiten können Sie Ihre Informationen überarbeiten. Nachdem Sie die notwendigen Korrekturen vorgenommen haben, sollten Sie die Änderungen Sichern, mit  Zurück zu Meine Biografie; gelangen Sie wieder zu Ihrer Biografie; Hauptseite.

Sie können jeden Ihrer Mitgliedschaften Einträge mit der Löschen Option entfernen.

Wenn Sie eine Beschreibung erfasst haben, zeigt Ihr Mitgliedschaften Eintrag einen Link, der die Beschreibung ausklappt oder zuklappt.
