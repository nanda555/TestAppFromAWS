Publications

Inscrivez ici toutes les publications que vous avez rédigées ou auxquelles vous avez pris part, par exemple an tant qu'éditeur ou illustrateur. Les données seront affichées dans un ordre chronologique inversé. Cliquez sur ‘Ajouter’ pour inscrire vos publications. Le bouton ‘Annuler’ fermera la section ‘Ajouter’ sans faire de sauvegarde.

Note: Il est important d'enregistrer vos données à l'aide du bouton situé sous cette section.

Les éléments nouvellement ajoutés s'afficheront les uns à la suite des autres dans votre CV. Modifiez votre texte en cliquant sur Modifier. N'oubliez pas d'enregistrer vos changements, puis cliquez sur Revenir au CV pour revenir à la page principale. Cliquez sur Supprimer pour effacer des éléments.

Si vous avez fourni une description de vos publications, cliquez sur le nom de la publication pour afficher ou cacher cette information.
