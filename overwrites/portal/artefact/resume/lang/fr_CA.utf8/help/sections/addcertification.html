Certifications, accréditations et récompenses

Inscrivez toutes les certifications, accréditations ou récompenses que vous avez reçues, par exemple: comptable agréé, infirmier autorisé, secouriste.

Les données seront affichées dans un ordre chronologique inversé. Cliquez sur ‘Ajouter’ pour inscrire vos certifications. Le bouton ‘Annuler’ fermera la section ‘Ajouter’ sans faire de sauvegarde.

Note: Il est important d'enregistrer vos données à l'aide du bouton situé sous cette section.

Les éléments nouvellement ajoutés s'afficheront les uns à la suite des autres dans votre CV. Modifiez votre texte en cliquant sur Modifier. N'oubliez pas d'enregistrer vos changements, puis cliquez sur Revenir au CV pour revenir à la page principale. Cliquez sur Supprimer pour effacer des éléments.

Si vous avez fourni une description de vos récompenses, cliquez sur le nom de la récompense pour afficher ou cacher cette information.
