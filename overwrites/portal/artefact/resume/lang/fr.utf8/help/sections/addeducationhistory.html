Historique de formation

Indiquez ici votre historique de formation et les certifications, diplômes et licences obtenus. Cliquez sur «Ajouter» pour créer une nouvelle entrée. Cliquez sur «Annuler» pour fermer le formulaire sans sauvegarder les informations saisies. Si vous avez plus d’une entrée, utilisez les flèches pour organiser leur ordre d’affichage.

Note : Il est indispensable de sauver chaque nouvelle entrée en utilisant le bouton au bas du formulaire.

Vos entrées sont affichées sous forme d’un tableau. Editez une ligne en cliquant sur Modifier. Quand vous avez fini, vous retournerez à cette page quand vous sauvez vos modifications. Cliquez sur Supprimer pour supprimer une ligne du tableau.

Si vous avez donné une description de la qualification obtenue, cliquez sur le nom de l’institution pour afficher ou masquer cette information.
