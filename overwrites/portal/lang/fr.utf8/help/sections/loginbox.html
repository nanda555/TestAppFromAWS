Connexion

Vos informations nécessaires à la connexion comprennent soit le nom et le mot de passe que vous avez donnés lors de votre enregistrement, soit ceux qui vous ont été indiqués par l'administrateur du site.

Cliquez sur Mot de passe oublié pour redemander un nouveau mot de passe en tout temps.

Pour des raisons de sécurité, vous serez automatiquement déconnecté après une période prolongée d'inactivité sur le site. Vous devrez alors vous authentifier à nouveau.

Note : L'utilisation de ce site, requière un logiciel de navigation web récent tel que : Internet Explorer version 7.0, ou n\'importe quelle version de Firefox, Safari, ou Chrome. Pour utiliser Mahara, votre navigateur doit accepter les cookies.
