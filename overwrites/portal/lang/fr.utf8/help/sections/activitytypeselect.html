Type d'activité

Sélectionnez dans le menu le type d'activité que vous désirez afficher.

Nouveaux messages de forums :affiche tous les nouveaux messages des forums auxquels vous avez souscrit un abonnement

Commentaires : affiche les commentaires qui ont été placés dans toutes vos productions, pages ou journaux dans votre portfolio.

Messages de groupe : affiche tous les messages envoyés aux gestionnaires de groupe quand une page est envoyée à l'évaluation à ce groupe.

Messages système : affiche les messages générés automatiquement par le système ainsi que ceux qui vous sont envoyés par les autres administrateurs.

Messages des autres utilisateurs : affiche les messages envoyés par les autres utilisateurs du site, directement à votre attention.

Nouvel accès à une page : affiche les alertes qui vous sont envoyées quand vous ou un groupe auquel vous appartenez, se voit donner l'accès à une page non publique, nouvelle ou existante.

Liste de suivi : affiche toutes les activités effectuées sur les productions, pages, groupes ou journaux pour lesquels vous avez activé le suivi.
