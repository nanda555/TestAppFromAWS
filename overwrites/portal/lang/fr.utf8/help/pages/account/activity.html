Boîte de réception

Il s'agit d'une liste d'événements ou d'activités du système qui vous concernent. Utilisez la Page de Notifications dans les Réglages pour indiquer comment vous désirez recevoir ces différents messages. Après avoir lu un message, vous pouvez le marquer pour suppression.

Note: Les messages seront automatiquement supprimés de votre liste après 60 jours que vous les ayez lus ou non.
