Rappel du nom d'utilisateur/mot de passe

Si vous avez oublié votre nom d'utilisateur ou votre mot de passe, le site peut vous envoyer un message vous indiquant votre nom d'utilisateur ainsi qu'un lien vous permettant de réinitialiser votre mot de passe. Entrez votre nom d'utilisateur ou adresse primaire de courriel dans le champ et cliquez sur « Envoyer la demande ». Vous recevrez un courriel automatiquement. Pour changer votre mot de passe, suivez le lien fourni dans le courriel pour confirmer votre demande de réinitialisation.

Attention: Si vous recevez un courriel de changement de mot de passe que vous n'avez pas sollicité, il se peut qu'il soit frauduleux. Ne cliquez en aucun cas sur un lien dans ce type de courriel.
