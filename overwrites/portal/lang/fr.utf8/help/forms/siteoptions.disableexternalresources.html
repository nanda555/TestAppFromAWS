Empêcher les ressources externes dans le code HTML de l'\utilisateur

En activant cette option, les utilisateurs ne pourront pas incorporer des images depuis d\'autres sites web dans leur code HTML.

Cette option est toutefois conseillée au niveau de la sécurité de votre site, afin de limiter les possibilités d'attaque par des systèmes intelligents de « phising »..

Référez-vous à la documentation de HTML Purifier pour plus d'informations.
