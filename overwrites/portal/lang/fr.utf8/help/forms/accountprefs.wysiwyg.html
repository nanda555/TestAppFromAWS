Editeur HTML

Un éditeur HTML est accessible pour la saisie simplifiée d'informations dans certaines sections de ce site. Cet éditeur est aussi commun comme éditeur WYSIWIG pour « What you see is what you get » (« Ce que vous voyez et ce que vous obtiendrez »). Cette fonction vous permet de formater votre texte comme dans un traitement de texte (ex. : italique, gras, retrait d'alinéa). Si l'éditeur est désactivé, vous pouvez toujours saisir du texte brut ou agrémenté de commandes HTML.

Cette fonction peut être désactivée par votre administrateur

Note : L'éditeur HTML peut s'afficher ou fonctionner de manières différentes en fonction du navigateur web que vous utilisez.
