Groupe public

Si vous répondez oui, la plupart des sections de votre groupe seront visibles par n'importe quelle personne visitant votre groupe - cela inclut les personnes qui n'ont pas de compte sur le site.

Tout le monde aura la possibilité de voir la liste des membres et toutes les discussions dans les forums. Par contre, pour pouvoir écrire dans les forums, il est nécessaire de devenir membre du groupe.

Quels que soient les paramètres choisis, la section fichiers de votre groupe et tous les fichiers inclus resteront privés.
