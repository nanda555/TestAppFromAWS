Afficher des avatars distants

Si cette option est activée, le service Gravatar sera utilisé pour afficher les avatars des utilisateurs qui n'auront pas téléchargé d'image dans leur profil. Ceci permet aux utilisateurs de placer une fois pour toutes un portrait dans Gravatar et de ne pas être obligés de télécharger, pour chaque site et service, une image pour afficher leur avatar. (Si Gravatar ne trouve pas d'image pour un utilisateur, c'est l'avatar donné par défaut qui sera affiché).

Raisons de ne pas utiliser cette option

Cette option devrait être désactivée pour des installations où le serveur n'a pas accès au réseau Internet (par exemple sur un réseau Intranet bloqué). Si Gravatar est activé sur des installations de ce type, cela ajoute des délais à l'affichage des pages pour rien.

De même si votre installation utilise le protocole protégé https, votre navigateur risque d'afficher régulièrement des alertes de sécurité, les images de Gravatar étant envoyée en mode non protégé par le protocole http standard.

Serveur Gravatar privé

Si vous hébergez un serveur qui comprend le protocole Gravatar, vous pouvez l'utiliser en donnant son url dans le fichier config.php :

$cfg->remoteavatarbaseurl = 'http://votreserveur.com/avatar/';
