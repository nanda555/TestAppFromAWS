Éditeur HTML

Vous pouvez définir si l’éditeur HTML doit être utilisé par défaut pour tous les utilisateurs du site. Si vous choisissez l’option Défini par l’utilisateur, les utilisateurs pourront choisir s’ils désirent ou non utiliser l’éditeur HTML. Dans le cas contraire, tous les utilisateurs se verront forcés à utiliser l’éditeur.
