Port SMTP

Si votre serveur SMTP utilise un port différent que le port 25, vous pouvez le spécifier ici.

Quand le serveur est en mode crypté, le port par défaut est le 465 pour SSL et 587 pour TLS. Vous devez spécifier le port seulement s’il est différent que ceux indiqués ici. Contrôlez vos paramètres avec votre prestataire de service de messagerie électronique.

Cette rubrique peut être désactivée si ces paramètres figurent déjà dans votre fichier de configuration « config.php ».
