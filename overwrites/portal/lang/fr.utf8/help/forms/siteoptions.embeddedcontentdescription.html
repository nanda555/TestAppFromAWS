Inclusion de contenu: sites sûrs

Pour éviter que du code malveillant venant d'autres sites web, les balises « objet » et « embed » sont normalement supprimées du code HTML avant d'afficher la page. Cette option permet toutefois aux utilisateurs d'ajouter ce type de code venant d'un certain nombre de sites reconnus sans danger, et de coller directement le contenu dans leurs pages.
