Contrôle anti-virus

Si vous désirez contrôler tous les fichiers téléchargés par les utilisateurs, vous devez sélectionner l'option de contrôle CLamAV anti-virus.

Vous devez avoir installé le logiciel gratuit ClamAV sur votre serveur. La plupart des distributions Linux proposent un paquet clamav, ou vous pouvez le télécharger depuis le site web de ClamAV.
