Thèmes

Quand plusieurs thèmes ont été installés, vous pouvez choisir depuis un menu déroulant, quel thème appliquer à votre site Mahara. Ce thème sera le thème par défaut pour tous les utilisateurs connectés.

Les Institutions peuvent choisir d'autres thèmes qui seront alors affichés lorsque les utilisateurs sont connectés à ces dernières. Vous pouvez configurer les thèmes que les Institutions peuvent utiliser, dans la section Gestion Institutions.
