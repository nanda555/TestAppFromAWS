Multi institutions

Lorsque cette option est activée, les utilisateurs peuvent appartenir à plus d'une institution en même temps. Les utilisateurs pourront alors demander leur affiliation à toutes les institutions qui le permettent. Leur demande devra bien entendu être validée par les gestionnaires de l'institution. D'autre part, les institutions pourront inviter les utilisateurs à les rejoindre; les utilisateurs devont alors accepter ou refuser cette invitation.
