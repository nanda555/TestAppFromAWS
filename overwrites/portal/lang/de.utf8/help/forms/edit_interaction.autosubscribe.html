Nutzer automatisch einschreiben

Wenn die Nutzer für die automatische Benachrichtigung eingetragen werden sollen, wenn sie dem Forum beitreten, setzen Sie die Option 'Ja'.

Die Nutzer erhalten so Benachrichtigungen wenn neue Nachrichten oder Antworten im Forum vorliegen.

Die Nutzer können sich abmelden, wenn sie nicht benachrichtigt werden wollen.
