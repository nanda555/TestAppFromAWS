Wöchentliche Updates verschicken

Wenn Sie die Option "Sende wöchentliche Updates" markieren, verschickt Mahara automatisch jede Woche eine Bericht mit aktualisierten Informationen über die Installation an mahara.org.

Die verschickten Inormationen sind high level statistische Daten - es werden keine persönliche Informationen über die Nutzer/innen verschickt. Sie können die Art der gesendten Daten vor der Registrierung überprüfen (dies geschieht über die Admin Home Seite).
