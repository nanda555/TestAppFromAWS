Tags

Sie können Ihre Ansichten verschlagworten. Mit diesen Tag können die Ansichten mit einer Beschreibung versehen werden, diese kann bei einer Suche verwendet werden.

Um ein Artefakt oder eine Ansicht mit mehreren Tags zu verbinden, trennen Sie die Tag-Einträge mit Kommata. Z.B.: "Praktikum, Lebenslauf, Schule". Der Klick auf Anzeige meiner Tags erzeugt eine Liste der Tags, die Sie vorher in Ihrem Portfolio benutzt haben. Die Zahl in den Klammern gibt die Häufigkeit an. Mit einem Klick auf einen existierenden Tag-Eintrag wird dieser zur Liste für das aktuelle Artefakt oder die Ansicht hinzugefügt. Mit der Funktion Mein Portfolio durchsuchen können Sie alle Artefakte und Ansichten finden, die Sie mit einem bestimmten Tag verknüpft haben.
