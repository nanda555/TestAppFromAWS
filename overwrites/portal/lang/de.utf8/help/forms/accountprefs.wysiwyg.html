HTML Editor

Der HTML Editor (auch als WYSIWYG Editor bezeichnet) ist in verschiedenen Bereichen der Site verfügbar. Der HTML Editor erlaubt ihnen die einfache Formatierung des erfassten Textes. Wenn der Editor ausgeschaltet ist, können sie nur unformatierten Text eingeben.

Beachten Sie bitte, dass der HTML Editor in den verschiedenen Browsern unterschiedlich aussehen und funktionieren kann.
