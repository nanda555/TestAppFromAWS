Externe Quellen für HTML Inhalte der Nutzer/innen deaktivieren

Wenn diese Option aktiviert ist, können die Nutzer/innen keine Bilder von fremdem Seiten in ihre Foren-Artikel und andere HTML Inhalte einfügen.

Aus Sicherheitsgründen ist dies aber eine gute Sache, da so einige Phishing Attacken abgewehrt werden können.

Besuchen Sie die Seite HTML Purifier Dokumentation für weitere Einzelheiten.
