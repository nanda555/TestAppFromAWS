Einrichtungen Ablaufdatum

Wenn Sie ein Ablaufdatum für eine Einrichtung festlegen, geschehen zwei Dinge. Wenn die Hinweiszeit für den Ablauf einer Einrichtung erreicht wird, erhalten die Website Administrator/innen und Einrichtungsadministrator/innen eine Nachricht über das anstehende Ablaufdatum.

Wenn die Option Automatische Sperrung abgelaufener Einrichtungen gesetzt ist, wird die Einrichtung automatisch mit Erreichen des Ablaufdatums gesperrt und die Nutzer/innen können sich nicht mehr einloggen.

Die Hinweiszeit für den Ablauf einer Einrichtung und die Option Automatische Sperrung abgelaufener Einrichtungen finden sich auf der Website Optionen Seite.
