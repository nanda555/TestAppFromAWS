Anzeige der Kontrollbuttons zum Hinzufügen und Löschen der Spalten in der Ansichtenbearbeitung

Sie können festlegen, ob eine Zeile mit den Buttons beim Bearbeiten einer Ansicht angezeigt werden. Mit diesen Buttons können Sie Spalten einfügen oder entfernen.

Wenn Sie diese Option hier ausschalten, können Sie die Spaltenanzahl immer noch bei Bearbeitung der Ansicht ändern.
