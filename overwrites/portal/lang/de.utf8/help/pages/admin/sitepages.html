Site Stammseiten

Der  Site Administrator kann die Stammseiten der Mahara Installation nach Bedarf bearbeiten. Wenn Sie die Stammseiten (Über, Startseite, Nutzerordnung, Copyright Notiz für das Hochladen, etc.) Ihren Bedürfnissen anpassen wollen, bearbeiten sie die gewünschte Seite mit dem  HTML Editor. Stellen Sie sicher, dass Sie den Text sichern, bevor Sie eine andere Seite aufrufen.
