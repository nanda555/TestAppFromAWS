Meine Gruppen

Gruppen sind eine Zusammenstellung von Nutzern, die gleiche Interessen haben

Ihre Gruppe Auflistungsseite zeigt alle Gruppen, bei denen Sie bestätigtes Mitglied sind

In Abhängigkeit von den Mitgliedschaftseinstellungen des Gruppenbesitzers können Sie normalerweise einer neuen Gruppe beitreten, indem sie die Startseite der Gruppe aufrufen. Die Gruppen-Startseite kann auch über die Funktion suche Gruppe aufgerufen werden.

Sie können jeder Gruppe, die Sie angelegt haben oder bei der Sie Mitglied sind, Zugriffsrechte für Ihre Ansichten gewähren. Wenn Sie einer Gruppe das Zugriffsrecht auf einen View gewähren, erhalten auch alle Mitglieder das entsprechende Zugriffsrecht.

Sie können eine Gruppe. die keine kontrollierte Mitgliedschaft durch den System-Administrator beinhaltet, jederzeit verlassen, indem Sie die Funktion Löschen wählen

Sie erreichen die Startseite einer Gruppe, indem Sie den Namen anklicken. Sie können dann eine Liste der Mitglieder und alle Links zu den verfügbaren Ansichten einsehen.

Meine eigenen Gruppen

Gruppen sind eine Zusammenstellung von Nutzern, die gleiche Interessen haben

Ihre Gruppe Auflistungsseite zeigt alle Gruppen, die Sie angelegt haben und die unerledigten Mitgliedschaftsanfragen an (dies geschieht in Abhängigkeit Ihrer gewählten Mitgliedschaftseinstellungen

Aus Gründen des Datenschutzes sehen Sie in der Auflistung nur die bevorzugten Namen der Gruppenmitglieder (die Admin und die Betreuer haben weitere Rechte)

Sie können jeder Gruppe, die Sie angelegt haben oder bei der Sie Mitglied sind, Zugriffsrechte für Ihre Ansichten gewähren. Wenn Sie einer Gruppe das Zugriffsrecht auf einen View gewähren, erhalten auch alle Mitglieder das entsprechende Zugriffsrecht.

Sie erreichen die Startseite einer Gruppe, indem Sie den Namen anklicken. Sie können dann eine Liste der Mitglieder und alle Links zu den verfügbaren Ansichten einsehen. Benutzen sie das Mitgliedschaft und Mitglieder Dropdown-Menü um die notwendigen Informationen zu bearbeiten und die Zugriffsrechte zu gewähren
