Geheime URL

Falls Sie Ihre Seite nicht öffentlich im Web zeigen, aber trotzdem für nicht registrierte NUtzer/innen von Mahara sichtbar machen wollen, so können Sie eine so genannte "geheime URL" erzeugen. Diese ist für Außenstehende schwer zu erraten, Sie können aber einzelnen Personen diesen Link zum Beispiel per E-Mail zusenden.
