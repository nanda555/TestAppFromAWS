Mein Portfolio durchsuchen

Mit dieser Funktion können Sie Ihre Tags, Beschreibungen, Namen, Titel oder Texte, die Sie innerhalb Ihrer Artefakte und Ansichten benutzt haben, durchsuchen. Das Suchergebnis liefert eine Linkliste zu allen entsprechenden Artefakten und Ansichten.

Sie können einen genauen Suchbegriff eingeben oder alle Artefakte und Ansichten innerhalb Ihres Portfolio durchsuchen, indem Sie ein leeres Suchfeld einreichen.

Indem Sie vom Tag-Feld und der Suche in meinem Portfolio Funktion Gebrauch machen, können Sie für Ihr Portfolio einen Verzeichnisindex anlegen
