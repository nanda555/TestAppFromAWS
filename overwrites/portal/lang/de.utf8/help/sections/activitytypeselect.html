Aktivitäten Typ

System Meldungen

Beinhaltet Meldungen, die vom System automatisch erzeugt wurden oder durch die Site Administratoren übermittelt wurden.

Meldungen von anderen Nutzern

Beinhaltet Meldungen, die Sie von anderen Nutzer/innen des Systems erhalten haben.

Feedback

Benachrichtigung über jegliche Feedbacks, die Sie für eine Ansicht oder Artefakt innerhalb Ihres ePortfolio erhalten haben. Die Kommentare, die Sie für verfügbare Blogs erhalten haben, werden auch als Feedback aufgeführt.

Beobachtungsliste

Benachrichtung über jegliche Aktivität bei jedem Artefakt, View oder Gruppe, das Sie über die Beobachtungsliste beobachten.

Neue Ansichten

Benachrichtigungen, dass Ihnen der Zugriff auf eine neue Ansicht erlaubt wurde, entweder als Person oder als Mitglied einer Gruppe oder Gruppe. Sie werden benachrichtigt, wenn eine öffentlich verfügbare Ansicht angelegt wird.

Neuer Ansichten Zugriff

Benachrichtigungen, dass Ihnen der Zugriff auf eine neue Ansicht erlaubt wurde, entweder als Person oder als Mitglied einer Gruppe oder Gruppe. Sie werden benachrichtigt, wenn ein öffentlich verfügbare Ansicht angelegt wird.

Administration Meldungen (nur Site Administratoren)

Benachrichtigungen und Hilfeanforderungen für Site Administratoren.
