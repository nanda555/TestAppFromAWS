Editor de HTML

Un editor HTML está disponible para su uso en algunas secciones del sitio. Es un editor de tipo "Lo que ves es lo que obtienes" (WYSIWYG). Esto le permite aplicar formato al texto, por ejemplo: negrita o cursiva. Si el editor está desactivado sólo se puede introducir texto.

Nota: Los editores de HTML pueden aparecer y funcionar algo diferente dependiendo del navegador.
